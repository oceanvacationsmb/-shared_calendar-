How To Update Locks

Use this when the calendar shows no locks or says 401 Not Authorized.

  1. Open Guesty linked locks while logged in.
  2. Press F12, open Network, then choose Fetch/XHR.
  3. Refresh Guesty and click the request that starts with locks?accountId=68d5ff6bea1895d9005ea411.
  4. Open Headers and copy authorization. Paste only the long token after Bearer into LOCKS_API_BEARER.
  5. If locks still fail, copy the cookie header from the same request into LOCKS_API_COOKIE.
  6. Click Save To Render. The API will redeploy and the calendar will use the new value.
