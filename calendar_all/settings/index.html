Where To Copy Locks Token

Use this only when you need a fresh LOCKS_API_BEARER.

  1. Open Guesty Locks while logged in.
  2. Press F12, open Network, then choose Fetch/XHR.
  3. Refresh Guesty and click the request that starts with locks?accountId=68d5ff6bea1895d9005ea411.
  4. Open Headers and copy authorization.
  5. Paste only the long token after Bearer into LOCKS_API_BEARER.
